# Light
A framework for coding desktop apps in javascript, html, and css <strong>(WIP)<strong/>

==========================================================

Dependencies:

PyQt5

PyQtWebEngine

EJS

Express

Python

NPM

==========================================================

Building:

git clone https://github.com/hugbot1/Light.git

cd Light

pip install PyQt5 PyQtWebEngine

npm init -y

npm install ejs

npm install express

==========================================================

Running:

node index.js

--Open a second terminal--

cd Light

python main.py
